### Core Result

Geodesics do not form a single generic route class on the PAM manifold.

In particular, **seam-contacting geodesics split into at least two distinct families**:

- **stable seam corridors**
- **reorganization-heavy seam routes**

This is the main result.

The seam is therefore not merely:
- a geometric divider
- a criticality-bearing region
- or an identity reorganization zone

It is also:

> **a routing structure that supports multiple distinct geodesic regimes**

---

### Why this matters

Earlier results established:

- OBS-013: identity compresses into angular type plus contextual axes
- OBS-014: the seam acts as an identity reorganization zone

OBS-015 adds the geometric routing layer:

> geodesics interacting with the seam do not all behave the same way

Some seam-contacting paths are:
- smooth
- corridor-like
- low-turnover

Others are:
- longer
- more articulated
- high-turnover
- strongly reorganization-bearing

That is a real structural refinement of the manifold.

### First Path-Level Result

Across the expanded path set, seam-contacting paths tend to show:

- higher criticality
- higher maximum criticality
- higher transport load
- more identity-sector turnover

Earlier diagnostics already indicated that seam-contact paths were not simply shortest routes that happened to pass nearby.

They preferentially traversed:

- more critical regions
- more loaded regions
- more identity-reorganizing regions

This motivated the family stratification pass.

### Path Family Stratification

Paths were stratified using:

- `near_fraction`
- `mean_angle_jump_deg`
- `n_sector_changes`

This yielded four interpretable route families:

1. **reorganization_heavy**
2. **stable_seam_corridor**
3. **off_seam_reorganizing**
4. **settled_distant**

These families were not degenerate and separated meaningfully in the summary statistics.
