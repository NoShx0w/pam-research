## Direct Definition

**Response-guided flow** is the family of discrete embedded trajectories generated by integrating the oriented dominant eigenvector of the local response tensor across the manifold node graph, under a forward-admissible step rule and optional seam-bundle scalar modulation.

## One-Line Version

A response-guided flow path is a discrete integral-like trajectory of the dominant response eigenvector field over the embedded manifold graph.

---

## 1. Base Manifold Object

Let the sampled manifold be a finite node set

\[
\mathcal{N} = \{x_i\}_{i=1}^N
\]

with each node carrying:
- parameter coordinates \((r_i,\alpha_i)\)
- embedded coordinates \(m_i \in \mathbb{R}^2\), typically
  \[
  m_i = (\mathrm{mds1}_i,\mathrm{mds2}_i)
  \]

The embedding is not the full manifold, but the working coordinate system in which first-pass flow is constructed.

## 2. Response Tensor Field

At each node \(x_i\), define the local response tensor

\[
T_i
=
\begin{pmatrix}
T_{xx}(x_i) & T_{xy}(x_i) \\
T_{yx}(x_i) & T_{yy}(x_i)
\end{pmatrix}
\]

This gives a nodewise \(2\times 2\) linear response operator over embedded local directions.

So the response field is the assignment

\[
T : \mathcal{N} \to \mathbb{R}^{2\times 2}.
\]

## 3. Dominant Response Direction

For each node \(x_i\), compute the eigensystem of \(T_i\).

If the eigenvalues are real and the eigensystem is valid, order them by descending absolute magnitude:

\[
|\lambda_1(x_i)| \ge |\lambda_2(x_i)|.
\]

The **dominant response direction** is the normalized eigenvector associated with \(\lambda_1\):

\[
v_i := \frac{e_1(x_i)}{\|e_1(x_i)\|}.
\]

Thus the response-guided direction field is

\[
v : \mathcal{N} \to S^1.
\]

### Direct definition

The dominant response direction at a node is the normalized eigenvector of the local response tensor with largest absolute eigenvalue magnitude.

### Interpretation

It is the locally preferred embedded direction of response.

## 4. Orientation Ambiguity and Oriented Local Flow

Because eigenvectors are defined only up to sign, \(v_i\) and \(-v_i\) represent the same eigendirection.

So response-guided flow uses a path-local orientation rule.

If a path arrives at node \(x_i\) with previous oriented direction \(u_{i-1}\), define the oriented local direction \(u_i\) by:

\[
u_i =
\begin{cases}
v_i & \text{if } \langle v_i, u_{i-1} \rangle \ge 0,\\
-v_i & \text{if } \langle v_i, u_{i-1} \rangle < 0.
\end{cases}
\]

At the seed node, choose:

\[
u_0 = v_0.
\]

### Direct definition

The oriented response direction is the sign-resolved dominant eigenvector chosen to maximize continuity with the previous step direction.

### Interpretation

This converts an unoriented eigendirection field into a pathwise flow direction.

## 5. Embedded Admissible Step Relation

Let \(m_i\) denote the embedded coordinate of node \(x_i\). For a current node \(x_i\), candidate next nodes are drawn from a local neighborhood:

\[
\mathcal{C}(i)
=
\left\{
j \neq i :
0 < \|m_j - m_i\| \le \rho
\right\}
\]

where \(\rho\) is the admissible step radius.

For each candidate \(x_j\), define the normalized displacement:

\[
d_{ij}
=
\frac{m_j - m_i}{\|m_j - m_i\|}.
\]

A candidate is **forward-admissible** if:

\[
\langle u_i, d_{ij} \rangle \ge c_{\min}
\]

where \(c_{\min}\) is the minimum forward cosine threshold.

Thus the admissible forward set is

\[
\mathcal{F}(i)
=
\left\{
j \in \mathcal{C}(i) :
\langle u_i, d_{ij} \rangle \ge c_{\min}
\right\}.
\]

### Direct definition

The forward-admissible next nodes are those lying within local step radius and sufficiently aligned with the oriented response direction.

### Interpretation

This is the discrete local analogue of stepping forward along a vector field.

## 6. Step Scoring and Flow Shaping

Among forward-admissible candidates, define a step score

\[
S(i \to j)
=
\langle u_i, d_{ij} \rangle
-
\beta \,\frac{\|m_j - m_i\|}{\rho}
-
\gamma \,\phi(x_j)
\]

where:
- the first term rewards directional alignment
- the second term mildly penalizes long hops
- \(\phi(x_j)\) is an optional flow-shaping scalar
- \(\gamma\) is the penalty weight

Examples of \(\phi\) used in OBS-043:
- `local_direction_mismatch_deg`
- `neighbor_direction_mismatch_mean`
- `node_holonomy_proxy`
- `lazarus_score`

The next node is chosen by

\[
x_{i+1}
=
\arg\max_{j \in \mathcal{F}(i)} S(i \to j).
\]

### Direct definition

The next step is the forward-admissible node maximizing the local response-flow score.

### Interpretation

Response-guided flow is not pure tangent following; it is tangent following under optional scalar modulation.

## 7. Discrete Response-Flow Path

Given a seed node \(x_0\), a **response-guided flow path** is the finite sequence

\[
\Gamma(x_0)
=
(x_0,x_1,\dots,x_K)
\]

constructed recursively by:
1. computing the oriented response direction \(u_k\)
2. constructing the forward-admissible set \(\mathcal{F}(k)\)
3. selecting the maximizing node \(x_{k+1}\)

The path terminates if:
- the maximum step count is reached, or
- \(\mathcal{F}(k)=\varnothing\)

In the current study, the dominant empirical termination mode is:
- forward-neighbor exhaustion

### Direct definition

A response-guided flow path is a finite embedded node sequence generated by repeated forward-aligned maximization under the oriented dominant response field.

### Interpretation

These are discrete integral-like curves of the response field.

## 8. Flow Observables

A response-guided flow path supports path-level observables.

### 8.1 Seam Engagement

If seam distance is \(d_{\mathrm{seam}}(x)\), define:

\[
\overline{d}_{\mathrm{seam}}(\Gamma)
=
\frac{1}{|\Gamma|}\sum_{x\in\Gamma} d_{\mathrm{seam}}(x)
\]

\[
d_{\mathrm{seam}}^{\min}(\Gamma)
=
\min_{x\in\Gamma} d_{\mathrm{seam}}(x)
\]

and seam contact indicator:

\[
\mathrm{SeamContact}(\Gamma)
=
\mathbf{1}\!\left[
d_{\mathrm{seam}}^{\min}(\Gamma)\le \tau_{\mathrm{seam}}
\right].
\]

This measures whether flow is seam-engaged.

### 8.2 Phase Interaction

If signed phase is \(\sigma(x)\), define:
- mean signed phase
- phase span

\[
\mathrm{PhaseSpan}(\Gamma)
=
\max_{x\in\Gamma}\sigma(x)-\min_{x\in\Gamma}\sigma(x)
\]

and phase-sign crossing:

\[
\mathrm{PhaseCross}(\Gamma)
=
\mathbf{1}\!\left[
\min_{x\in\Gamma}\sigma(x) < 0 < \max_{x\in\Gamma}\sigma(x)
\right].
\]

This measures whether flow traverses phase structure.

### 8.3 Path Extent

In embedded coordinates, path length is

\[
L(\Gamma)
=
\sum_{k=0}^{K-1}
\|m_{k+1}-m_k\|.
\]

Step count is:

\[
K = |\Gamma|-1.
\]

This measures how continuous the flow can remain under the discrete integrator.

### 8.4 Flow-Shaping Scalar Exposure

If \(\phi(x)\) is the active penalty scalar, define:

\[
\overline{\phi}(\Gamma)
=
\frac{1}{|\Gamma|}\sum_{x\in\Gamma}\phi(x),
\qquad
\phi_{\max}(\Gamma)
=
\max_{x\in\Gamma}\phi(x).
\]

This measures what scalar regime the path traverses.

## 9. Response-Guided Flow Field

The total response-guided flow construction on the sampled manifold is the pair

\[
(\mathcal{N}, v)
\]

together with:
- orientation rule
- admissible step relation
- step score
- termination law

The resulting path family from a seed set \(S \subseteq \mathcal{N}\) is

\[
\mathfrak{F}(S)
=
\{\Gamma(x) : x\in S\}.
\]

### Direct definition

Response-guided flow is the family of discrete integral-like trajectories induced by the dominant response eigenvector field over the embedded node graph.

### Interpretation

It is the first observatory dynamical layer derived from the response tensor field.

## 10. Empirical Stabilization from OBS-043

OBS-043 stabilizes the following empirical claims about this object.

### 10.1 Local coherence

The response-guided direction field exhibits coherent local patches and is not noise.

### 10.2 Strong seam engagement

Flow remains consistently high in seam engagement across tested regimes.

### 10.3 Dual seam behavior

The seam behaves both as:
- local capture structure
- launch region for broader response-guided routing

### 10.4 Phase permeability

Under relaxed integration, flow increasingly crosses phase sign.

### 10.5 Scalar modulation

Seam-bundle mismatch scalars modulate routing behavior without collapsing seam engagement.

In particular:
- weak local mismatch penalties behave more like continuity regularizers than hard barriers
- neighborhood directional mismatch behaves more like a routing cost than a seam-repulsion barrier

## 11. Current Limitation

The current object is a **discrete embedded flow**, not a continuous flow reconstruction.

Its present limitations are:
- finite node graph
- neighborhood-radius dependence
- forward-cos threshold dependence
- termination by forward-neighbor exhaustion
- pathwise orientation rather than globally oriented field integration

So the current object is best described as:

\[
\text{response-guided discrete flow}
\]

not yet:
- continuous response flow
- transport PDE
- or globally smooth integral-curve dynamics

The key protective conclusion from OBS-043 is:

**the current limitation is methodological, not conceptual.**

## 15. OBS-043 Current Route-Family Refinement

Follow-on analysis of the OBS-043 flow outputs resolves response-guided flow into a first-pass route-family taxonomy.

Current route families are:
- `seam_hugging`
- `release_directed`
- `short_trapped`
- `mixed`

A key refinement is that:
- **cross-phase is not treated as a route family**
- it is treated as a **path attribute**

This correction matters because early classification attempts over-assigned seam-hugging paths into a separate `cross_phase` class, obscuring the actual route structure.

### Current empirical reading

The current first-pass reading is:
- `short_trapped` is the largest single route family under the present discretized integrator
- `release_directed` is a stable minority family
- `seam_hugging` is also a stable family once cross-phase is treated as an attribute rather than an overriding class

### Scalar modulation refinement

Neighborhood directional mismatch produces the clearest route-family-level modulation currently observed.

Its most important effect is:
- preservation of seam-engaged route families
- reduction of cross-phase release behavior
- modest shortening of path extent

This supports the current interpretation that neighborhood directional mismatch behaves more like a routing cost than a seam-repulsion barrier.

### Current limitation

The route-family picture remains conditioned by the current discrete node-hopping integrator.

In particular:
- all tested regimes still terminate by forward-neighbor exhaustion
- the current taxonomy should therefore be read as a first-pass discretized routing classification rather than a final continuous-flow decomposition
