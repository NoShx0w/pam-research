# Observatory Interface

This directory is the canonical home of the PAM observatory interface.

It owns the next-generation observatory TUI / GUI implementation.

Design principles:
- manifold-centered
- file-first
- one shared selected node
- multiple structural lenses
- identity mode centered on holonomy and transport-derived local obstruction

Legacy interface code has been archived from prior locations such as:
- `tui/`
- `observatory_ui/`
